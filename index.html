Trello Booking App

Trello Booking App is a generic booking app that could be used for booking hotels, car rentals, flight tickets, etc. It gives users the privilege to comment on bookings that they have visited previously and rate it. Visitors can see ratings, description ,features of that booking. Users can also view comments of registered users.

Features
  • Server Side Caching
  • Chat Messenger
  • Photo Uploads
  • Comment and rate a booking
  • Manage User/Admin Roles
  • Responsive Web App
